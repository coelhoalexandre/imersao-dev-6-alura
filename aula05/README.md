## Visão Geral

Projeto realizado enquanto assistia a aula 05 da Imersão Dev 6 e posteriormente modificado por mim.

Enquanto este ReadMe está sendo escrito, a aula ainda está liberada. Pode ser acessada através deste link: <a href="https://imersao.dev/aulas/aula05-tabela-pontos">Imersão Dev 6 - Aula 05</a>.

## Modificações

- Troca do atributo **onclick** pelo método **addEventListener**;
- Adição de um input que coleta nomes;
- Botão que adiciona os nomes na tabela;
- Botão que apaga toda a lista de jogadores atuais da tabela;
- Botão que reinicia a tabela, com os mesmos jogadores, entretanto sem nenhuma pontuação.
